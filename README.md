# RentGo Black Gold Luxury

## Deskripsi

RentGo Black Gold Luxury merupakan sistem informasi rental mobil berbasis web yang dikembangkan untuk mempermudah proses penyewaan kendaraan secara online. Sistem ini menyediakan fitur pemesanan mobil, pembayaran, verifikasi oleh admin, invoice, riwayat penyewaan, dan dashboard administrasi.

---

## Teknologi

- PHP Native
- MySQL / MariaDB
- Bootstrap 5
- HTML5
- CSS3
- JavaScript

---

## Fitur

- Login & Register
- Dashboard Admin
- Dashboard User
- CRUD Data Mobil
- Booking Mobil
- Upload Bukti Pembayaran
- Verifikasi Pembayaran
- Invoice
- Riwayat Booking
- Review Pelanggan

## Instalasi

1. Jalankan Apache dan MySQL melalui XAMPP.
2. Import database `database/rentgo.session.sql`.
3. Buka browser.
4. Akses:

```
http://localhost/RentGo
```
